# Проект "HeadHunterVacancyDataBase"

## Описание:

HeadHunter Vacancy Database — инструмент для сбора, хранения и анализа данных о вакансиях ведущих компаний России 
на платформе HeadHunter.ru.

Проект позволяет:

- Получать актуальные вакансии 10 крупнейших работодателей России (Avito, Точка Банк, Купер, Aston, VK, Контур, 
Тензор, Яндекс, 2ГИС, Skyeng).

- Сохранять информацию о компаниях и вакансиях в базе данных PostgreSQL.

- Выполнять аналитические запросы через удобное меню CLI.

- Проект реализован с использованием ООП-подхода, модульной архитектуры и автоматического обновления данных 
через API HeadHunter.

## Функциональность

### Работа с API HeadHunter

- Абстрактный класс BaseAPISource и классы-наследники HeadHunterVacanciesSource, HeadHunterEmployersSource.

- Получение данных о компаниях и их вакансиях через публичное API.

- Обеспечение актуальности информации при каждом запуске программы.

### Создание и работа с базой данных

- Класс HeadHunterDataBase отвечает за подключение к PostgreSQL, создание таблиц hh_companies и hh_vacancies.

- Автоматическое заполнение базы актуальными данными при запуске.

- Возможность пересоздания базы для обновления информации.

![img.png](data/img.png)

### Взаимодействие с пользователем

- CLI-интерфейс через класс UserInteraction.

- Пользователь выбирает запрос из меню и получает данные в удобной форме.

````
# Пример вывода пользователю:

Добрый день!
Идет загрузка данных ...

Пожалуйста, выберите пункт меню:
    1. Список компаний и количество вакансий 
    2. Информация обо всех вакансиях
    3. Средняя зарплата по вакансиям у каждой компании
    4. Список вакансий с зарплатой выше средней
    5. Список вакансий по ключевому слову в названии
    6. Выйти из программы

````

### Обработка данных

- Класс HeadHunterDataCoordinator управляет запросами к базе данных и форматирует результаты.

- HeadHunterDataBaseManager выполняет SQL-запросы и возвращает данные в виде структурированных списков.

- VacancyManager фильтрует, сортирует и обрабатывает вакансии по диапазону зарплат и ключевым словам.

- Табличный вывод реализован через библиотеку tabulate.

## Установка:

1. Клонируйте репозиторий проекта:
````
git clone https://github.com/simbarilion/HH_Vacancy_DB.git

cd HH_Vacancy_DB
````
2. Установите зависимости внутри каталога проекта для создания виртуального окружения и установки зависимостей:
````
poetry install
````
3. Настройте соединение с PostgreSQL через .env (при необходимости)
